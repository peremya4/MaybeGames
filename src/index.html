MaybeGames

Small game factory

This repository is a factory for tiny, experimental browser games. Each prototype lives in its own folder under games/ so new ideas can land without disturbing unrelated work.

What lives here

  • src/ — this home page and any shared static shell.
  • games/<game-slug>/ — one self-contained game per folder.
  • docs/ — how we work (including the AI pipeline).

Next steps

Open src/index.html locally or run npm start for a simple static server. When you add a game, follow the convention in games/README.md.

Plain HTML, CSS, and JavaScript — no backend required for the shell.
