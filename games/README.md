# Games

Each folder is a self-contained browser prototype (HTML, CSS, JS).

| Slug | Title | One line |
|------|-------|----------|
| [reflex-dodge](reflex-dodge/) | Reflex Dodge | Move, dodge red blocks, grab green orbs. |

Add new games under `games/<slug>/` with `index.html`, `main.js`, `styles.css`, and a `README.md`.
